This project is a Book Reselling System developed using Python for the backend and MySQL as the database management system. It provides a platform for users to list, buy, and sell used books efficiently. The system supports user registration, book catalog management, and transaction handling, all integrated with a MySQL database to securely store user and book information. This project showcases CRUD operations, user authentication, and database connectivity using Python and MySQL. It is designed to facilitate peer-to-peer book exchange and aims to simplify the process of reselling books within a community
